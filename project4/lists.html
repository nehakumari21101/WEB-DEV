What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of a webpage using elements and tags.

Example:

<!DOCTYPE html>

<html>

<head>

  <title>My First Page </title>

</head>

<body>

  <h1>Hello World! </h1>

</body>

</html>
