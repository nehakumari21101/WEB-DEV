HTML Tables

HTML tables allow web developers to arrange data into rows and columns.

A table in HTML consistsof table cells inside rows and columns.

Example:

<!DOCTYPE html>

<html>

<body>

<table>
  <tr>
    <th>Roll No</th>
    <th>Name</th>
    <th>Course</th>
  </tr>

  <tr>
    <td>101</td>
    <td>Neha Kumari</td>
    <td>Web Development</td>
  </tr>

  <tr>
    <td>102</td>
    <td>Rohan Verma</td>
    <td>Data Science</td>
  </tr>

  <tr>
    <td>103</td>
    <td>Priya Singh</td>
    <td>UI/UX Design</td>
  </tr>

  <tr>
    <td>104</td>
    <td>Amit Patel</td>
    <td>Python Programming</td>
  </tr>

  <tr>
    <td>105</td>
    <td>Riya Sharma</td>
    <td>Artificial Intelligence</td>
  </tr>

</table> </body>

</html>
Roll No Name Course
102 Rohan Verma Data Science
103 Priya Singh UI/UX Design
104 Amit Patel Python Programming
105 Riya Sharma Artificial Intelligence

Table Cells

Each table cell is defined by a <td> and a </td> tag.

td stands for table data.

Everything between <td> and </td> is the content of a table cell.

<table>
  <tr>
    <td>Enil</td>
    <td>Tobbias</td>
  </tr>

  <tr>
    <td>16</td>
    <td>14</td>
  </tr>

</table>

Note: A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.


Table rows

Each row starts with a <tr> and and ends with </tr>

tr stands for table row

<table>
  <tr>
    <td>101</td>
    <td>Neha Kumari</td>
  </tr>
  <tr>
    <td>102</td>
    <td>Rohan Verma</td>
  </tr>
</table>

Table Headers

Sometimes you want your cells to be table header cells. In those cases use the <th> tag instead of the </th> tag:

th stands for table header.

<table>
  <tr>
    <th>Person 1</td>
    <th>Person 2</td>
  </tr>
  <tr>
    <td>101</td>
    <td>Neha Kumari</td>
  </tr>
  <tr>
    <td>102</td>
    <td>Rohan Verma</td>
  </tr>
</table>

By default, the text in elements are bold and centered, but you can change that with CSS.
